# Odoo Studio Decision

## Kurzentscheidung

`Odoo Studio` ist kein kostenloser allgemeiner Baukasten, mit dem sich beliebig viele selbst gehostete Apps gratis bauen und betreiben lassen.

## Einordnung

- Laut Odoo gehoert `Studio` in den `Custom`-Pfad.
- `One App Free` ist kein Argument fuer einen allgemeinen selbst gehosteten App-Baukasten mit Studio.
- Studio ist ein optionales Business-Werkzeug fuer Odoo-Anpassungen, nicht die Grundlage der Plattformstrategie.

## Konsequenz fuer FRAWO

- Odoo bleibt wichtig fuer ERP/CRM/Rechnungen.
- Odoo Studio kann spaeter fuer gezielte Formulare, Felder oder interne Prozesse sinnvoll sein.
- Website, Mail, Secrets, Support- und Infrastrukturpfade werden nicht von Odoo Studio abhaengig geplant.

## Default

- vor Release keine Studio-getriebene Zusatzkomplexitaet
- Studio spaeter nur dann, wenn ein klarer Business-Use-Case mit Budget und Pflegepfad existiert
